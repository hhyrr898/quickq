Windows 后台东西多，断线原因经常不是 quickq 本身。我测试时在 Win11 23H2 上跑 2.8.4，发现开着公司杀毒软件的「网络防护」时，每小时必断一次，加白名单就好了。

## 五个 FAQ

### Q1：连不上，第一步干啥？

看别的网站能不能开。整机都上不了网，重启路由或换热点。只有 quickq 不行再查客户端。

### Q2：防火墙要咋设？

Windows 安全中心 → 防火墙 → 允许应用，把 quickq 勾上专用和公用网络。第三方杀毒同理加白名单。测试完记得别开永久关防火墙。

### Q3：线路怎么选？

客户端里看延迟数字，选低的。换完等 15 秒再开网页，别连点三条。

### Q4：版本旧会有问题吗？

要 2.8.4（2026年6月）。设置里检查更新，或下载中心下新包覆盖装。

### Q5：账号安全顺带做啥？

独立强密码，设备管理看不认识的机子就踢。钓鱼邮件里的「quickq 登录」链接别点。

## 额外三步操作

**第 1 步：** 管理员身份运行客户端（右键图标 → 以管理员身份运行）  
**第 2 步：** 关网盘同步和大下载再连  
**第 3 步：** 记录断线时间，如果总在晚 8 点，可能是小区宽带高峰

## 常见问题

**客户端闪退？**  
完全卸载后从下载中心重装 2.8.4，别留旧配置文件夹。

**连上托盘是绿的但网页超时？**  
等 20 秒或断开重连，DNS 缓存偶尔捣乱， `ipconfig /flushdns` 也行。

**公司电脑策略拦了？**  
个人热点对比：热点行就是公司网问题，找 IT。

**需要重装系统吗？**  
不用，上面都试过再考虑。多数是白名单或线路的事。
