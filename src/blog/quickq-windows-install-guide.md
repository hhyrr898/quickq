## 安装前准备

Windows 用户开始安装 quickq 前，建议先确认系统版本、剩余空间、浏览器下载状态和账号信息。常见的 Windows 10、Windows 11 都适合日常使用，但如果电脑长期没有更新，安装过程中可能会遇到权限提示或运行库提示。安装前关闭大型下载、网盘同步和占用网络的会议软件，可以让首次启动更顺畅。若电脑由公司统一管理，还应先确认是否允许安装新客户端，避免安装到一半被系统策略拦截。

## 安装流程

获取 quickq Windows 版安装包后，双击打开安装程序。如果系统弹出安全确认窗口，先核对文件名称，再继续下一步。安装位置通常保持默认即可，除非系统盘空间很小，才需要改到其他磁盘。安装完成后，可以从桌面图标、开始菜单或任务栏搜索进入 quickq。首次打开时不要急着切换多个设置，先让客户端完成初始化，再查看主界面状态。

## 首次登录

登录前确认电脑时间准确，账号和验证方式可用。若 quickq 提示需要网络权限，建议按系统弹窗完成允许，否则连接状态可能无法正常显示。登录后可先查看账号信息、设备名称和默认线路，不建议在不熟悉的情况下连续更改多项参数。新设备首次使用时，保持默认配置通常更容易判断是否安装成功。

## 连接确认

首次连接完成后，可以打开常用网页、工作系统或资料页面做简单检查。若页面响应较慢，先等待十几秒，再切换线路或重新连接。很多问题并不是客户端本身造成的，路由器信号、浏览器缓存、系统后台任务都会影响体验。建议记录首次安装时的系统版本、客户端版本和网络类型，后续排查会更清楚。本文下方的视频区域可配合阅读，适合作为 Windows 安装视频教程的页面使用。
