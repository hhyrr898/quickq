## 设备确认

Mac 用户安装 quickq 前，先在“关于本机”里查看系统版本和芯片类型。Apple 芯片与 Intel 芯片的日常使用方式相近，但首次打开应用时，系统提示可能略有差异。若设备长期未更新，建议先完成必要的系统更新，再下载安装包。安装前还可以检查存储空间和网络状态，避免下载完成后因为空间不足或网络中断导致安装失败。对于经常移动办公的用户，建议在稳定 Wi-Fi 下完成第一次配置。

## 权限设置

安装完成后，macOS 可能要求确认网络、通知、后台运行或登录项权限。遇到提示时，不要直接关闭窗口，可根据 quickq 页面说明进入系统设置逐项确认。若权限没有完成，客户端可能能打开，但连接状态、通知提醒或后台保持会受到影响。对于新手来说，最稳妥的方式是先完成基础授权，再启动连接测试。

## 菜单栏使用

quickq macOS 版常用状态通常会显示在菜单栏，适合快速查看连接、切换线路和进入设置。若菜单栏图标没有出现，可以先退出客户端再重新打开，也可以检查系统是否隐藏了菜单栏图标。日常使用时，菜单栏入口比反复打开主窗口更方便，尤其适合写作、会议和资料查阅场景。

## 日常维护

保持系统时间准确、定期检查客户端版本、减少无关后台任务，是 Mac 上保持稳定体验的基础。遇到连接异常时，先确认本地网络是否正常，再查看 quickq 状态。如果刚刚切换过 Wi-Fi 或热点，建议等待系统完成网络重连后再操作。对于多台 Mac 或 iPad 配合使用的用户，可以统一设备命名，后续查找登录状态会更清楚。
